# Nexora - Static Web Site 🚀
> **Note:** This README was written with the help of AI 🤖

Nexora is a **responsive static website** showcasing web services.  
The site is built using **pure HTML and CSS**, without any frameworks or JavaScript.


## ✨ Features
- **Why Choose Nexora Section**: Stylish hover effects on cards.  
- **Services Section**:  
  - Hover effects on service cards.  
  - Click "Learn More" to open detailed service pages.  
- **Colleagues' Projects**: Explore projects created by my colleagues.  
- **Contact Form**: Validation messages for valid and invalid inputs.  
- **Responsive Design**: Works perfectly on all screen sizes (mobile, tablet, desktop).

## 🛠 Tech Used
- **HTML5**
- **CSS3** (pure, no frameworks)
